flechaMedio()

flechaMedio(string  $cadena) : boolean

Cadena "<->" en medio en un cojunto de caracteres.

Recivimos una $cadena, y tenemos que comprobar si la siguiente serie "<->", aparece en medio. Se considerará que está en medio si el número de caracteres que hay antes y después difieren como mucho en una unidad. devolverenos TRUE de lo contrario FALSE.

Parameters

string $cadena

donde recibimos la cadena string.

Returns

boolean —

devuelve un True si es correcto y False si no se comple la condición.

Examples

('asd<->asd'). Se comprueba si <-> estan en medio. Si es que no se devolvera False De lo contrario se comprobara si a los lados se encuentra <> Si alguno de estos dos elementos no estuviera en la posicion correcta se devolveria False si no devolvera TRUE. Tambien se considera que esta en medio cuando sea una posicion menos y mas por la posibilidad de que el medio pudiera ser impar.
